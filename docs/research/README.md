## Research Keywords

When searching for related research, use these keywords:

### Primary Keywords
- `multi-agent debate LLM`
- `LLM debate reasoning`
- `multi-LLM collaboration`
- `AI agent debate`
- `society of mind LLM`

### Secondary Keywords
- `LLM self-correction`
- `multi-agent reasoning`
- `collaborative AI systems`
- `LLM ensemble methods`
- `adversarial collaboration AI`

### Academic Venues
- **NeurIPS** - Neural Information Processing Systems
- **ICML** - International Conference on Machine Learning
- **ACL/EMNLP** - Association for Computational Linguistics
- **ICLR** - International Conference on Learning Representations
- **arXiv cs.CL** - Computation and Language preprints

## How to Contribute Research

1. Find relevant papers using the search queries above
2. Add to [PAPERS.md](./PAPERS.md) with:
   - Title, authors, year
   - arXiv/DOI link
   - Key contribution summary
   - Relevance to Obora
3. Update [TRENDS.md](./TRENDS.md) if new research directions emerge
