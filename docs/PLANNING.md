# obora 기획서

## 프로젝트 비전

> **여러 상용 AI 코딩 도구를 하나의 세션에서 통합하고, AI들이 서로 토론하며 협업하는 환경**

---

## 핵심 컨셉

### 상용 AI 통합

```
기존: 각 AI 도구를 따로 실행, 결과를 수동으로 조합
obora: 여러 AI를 하나의 세션에서 통합 → AI끼리 토론/협업
```

### 지원 대상 AI (API/헤드리스)

- Claude Code
- Cursor Agent
- GitHub Copilot / Codex
- Gemini
- 기타 API 제공 AI 도구들

### 예시: 코드 리뷰 세션

```
You: 이 PR 리뷰해줘

[Claude Code]: 전체 구조를 보면, 이 부분이 복잡해 보여요...

[Cursor]: 동의해요. 그리고 여기 타입 에러 가능성이...

[Copilot]: 테스트 관점에서 보면 이 케이스가 빠졌어요...

[Claude Code]: 맞아요, Copilot 말대로 테스트 추가하고,
Cursor가 지적한 타입도 수정하면 좋겠네요.

→ AI들이 합의한 리뷰 결과 제공
```

## 핵심 가치

| 기존 방식 | obora |
|----------|-------|
| 하나의 AI만 사용 | 여러 상용 AI 동시 활용 |
| 각 도구 따로 실행 | 하나의 세션에서 통합 |
| 결과만 받음 | AI끼리 토론하고 의사결정 |
| 수동으로 결과 조합 | 자동으로 협업 결과 도출 |

## 협업 모드

### 1. Sequential (순차)
```
You → Claude Code → Cursor → Copilot → 결과
```
- 순서대로 작업 전달
- 이전 AI의 출력이 다음 AI의 입력
- 용도: 파이프라인 (분석 → 수정 → 테스트)

### 2. Parallel (병렬)
```
You → Claude Code ─┐
    → Cursor      ─┼→ 결과 비교
    → Copilot     ─┘
```
- 동시에 같은 질문
- 각 AI의 관점 수집
- 용도: 다각도 리뷰, 의견 수집

### 3. Debate (토론)
```
Round 1: Claude Code → Cursor → Copilot
Round 2: Claude Code → Cursor → Copilot
→ 종합 결론
```
- 서로의 의견에 반응
- 라운드별 토론
- 용도: 기술 의사결정, 아키텍처 논의

### 4. Autonomous (자율 협업) ⭐ 핵심
```
You: 질문/요청

AI들이 자유롭게:
- 발언할 타이밍 스스로 판단
- 다른 AI 의견에 반응
- 필요시 끼어들기
- 합의점 도출

→ 실제 회의처럼 자연스러운 협업
```

## 사용 시나리오

### 1. 코드 리뷰
```
You: 이 코드 리뷰해줘

[Claude Code]: 로직 분석 결과...
[Cursor]: 리팩토링 제안...
[Copilot]: 테스트 케이스 제안...

→ 통합된 리뷰 피드백
```

### 2. 아키텍처 논의
```
You: 이 서비스 아키텍처 어떻게 구성할까?

[Claude Code]: 마이크로서비스가 좋을 것 같아요...
[Gemini]: 동의하지만, 이 규모에선 모놀리스도...
[Claude Code]: 좋은 지적이에요, 그럼 모듈러 모놀리스는?
[Gemini]: 그게 적절해 보여요.

→ 합의된 아키텍처 결정
```

### 3. 버그 해결
```
You: 이 버그 원인 찾아줘

[Cursor]: 이 라인이 의심되는데...
[Claude Code]: 동의해요, 그리고 여기도 관련있어 보여요
[Copilot]: 비슷한 패턴으로 다른 곳에도 버그 있을 수 있어요

→ 종합적인 버그 분석과 해결책
```

## 사용자 인터페이스

### 명령어

```
/ai <name>         특정 AI에게 직접 요청
/ais               사용 가능한 AI 목록
/mode <mode>       협업 모드 변경 (sequential/parallel/debate/autonomous)
/end               토론 종료
```

## 결론

obora는 **여러 상용 AI를 하나의 세션에서 통합**하고,
**AI들이 서로 토론하며 협업**하는 환경을 제공합니다.

```
"하나의 AI에게 물어보기"
    → "여러 AI에게 동시에 물어보기"
        → "AI들이 서로 토론하고 합의하기"
```

핵심 차별점: **AI간 자율 협업** - 여러 상용 AI가 하나의 세션에서 토론하고 의사결정
